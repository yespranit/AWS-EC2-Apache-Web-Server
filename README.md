\# AWS EC2 Apache Web Server



\## Project Overview



This project demonstrates how to launch an Amazon Linux EC2 instance on AWS and host a simple website using the Apache HTTP Server.



\---



\## Objective



\- Launch an EC2 instance

\- Connect using SSH

\- Install Apache HTTP Server

\- Deploy a static HTML website

\- Verify the website in a browser



\---



\## AWS Services Used



\- Amazon EC2

\- Security Groups

\- Key Pair

\- VPC (Default)

\- Internet Gateway



\---



\## Software Used
